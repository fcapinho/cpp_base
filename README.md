# C++ Base Project

An initial base multi-platform C++ project with [GoogleTest](https://github.com/google/googletest) unit tests.

## Windows

###  Build

`cmake --build .`

or

`msbuild Base.sln`

### Test

`ctest -C "Debug"`

or 

`Debug\base_test.exe`
